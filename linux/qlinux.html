Qué es Linux?

Linux es un Sistema Operativo (OS) del tipo Unix que fue desarrollado por un estudiante finlandes llamado Linus Torvalds (capo). Empezo su tarea en el anio 1991 y en 1994 terminó su version 1.0 del Linux Kernel. El desarrollo continua constantemente y la version actual es la 2.4.
Linux se desarrolla bajo la Licencia Publica General GNU y su codigo fuente se encuentra disponible gratuitamente para todo el mundo.
Linux puede ser usado para una variedad de propósitos, incluyendo redes, desarrollo de software y como plataforma para el usuario final. Es considerado una excelente alternativa de bajo costo que otros sistemas operativos.


 El pinguino Tux es la mascota oficial de Linux. Fue seleccionado por Linus Torvalds para representar la image que él asocia con el sistema operativo que creó.

Y por que usar Linux?

La mayoria de la gente que esta leyendo este articulo esta probablemente usando algun tipo de Windows. Si sos de este tipo de lectores y solo usas tu computador para hacer procesamiento basico de textos o hojas de calculo, te recomiendo que sigas usando tu Windows; con todas sus fallas, es facil de usar, relativamente facil de aprender y tiene buen software.
Para todos los demas, sigan leyendo.
Aqui hay diez razones porque Linux podria ser el mejor sistema operativo para vos:

  • Una distribucion de Linux (los CDs de instalacion) tienen miles de dolares de software sin costo.
  • Linux es un sistema operativo completo que es:
    • estable: si un programa se 'cae' es mucho mas improbable que haga caer todo el sistema
    • confiable: los servidores Linux a menudo estan en servicio por cientos de dias, comparados con los reboots requeridos con un sistema Windows.
    • extremadamente poderoso
  • Incluye un entorno completo de desarrollo, incluyendo lenguajes C, C++, compiladores Fortran, herramientas y lenguajes encriptados como Perl, awk y sed. Solo un compilador para Windows cuesta cientos de dolares.
  • Excelente soporte para redes
  • El entorno ideal para ejecutar servidores web como Apache o servidores FTP.
  • Facilmente actualizable. Despues de un tiempo, una tipica instalacion de Windows se vuelve un desastre, con muchos conflictos que se revelan como una baja en la performance del equipo. ("esta muy lenta"). Linux es mucho mejor para mantener el sistema.
  • Una amplia variedad de software comercial esta disponible si tus necesidades no son satisfechas por el software gratis.
  • Un excelente sistema de ventanas llamado X; el equivalente de Windows pero mucho mas flexible.
  • Soporta multiples procesadores como estandar.
  • Verdadera multi-tarea: la habilidad de ejecutar mas de un programa al mismo tiempo.

Por supuesto hay muchas otras razones para usar Linux. Es posible preparar su sistema para que tenga mas de un sistema operativo. Si estas contemplando usar Linux, esto puede ser muy util. Te dejaria quedarte con Windows y usar Linux. Esto requerira crear particiones en tu disco duro. Las distribuciones de Linux explican como crear un sistema multi-boot.

Algunos mitos sobre Linux y mi experiencia personal.

* Solo personas de alto conocimiento en computacion pueden mantener el sistema actualizado.
- En RedHat y SuSE, hay una opcion para actualizar el sistema con un par de botones y automaticamente.

* Cambiarme a Linux significa tener que perder todas mis cosas de Windows.
- Mi primera instalacion ocupo solo una particion del disco, manteniendo intacta mi particion de Windows. Asi puedo bootear en Linux o Windows, segun mi necesidad (la unica razon para hacerlo es para grabar un cd porque mi cd-r no graba en Linux, el fabricante no lo soporta). Desde el laptop en win puedo leer mi Linux file server e imprimir con Samba.
Linux Myths

Por ultimo, Linux esta mejorando. No es una empresa la que lo produce y vende. Son miles de programadores dispersos por todo el mundo que lo mejoran cada dia. Desde mi primera instalacion adonde tuve que 'tocar' un poco de todo, hasta mi ultima instalacion, te puedo decir que Linux mejora muchisimo. No te sorprenda que en dos o tres años esté al mismo nivel tecnico para el usuario final que el monstruo de Microsoft.

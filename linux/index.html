Un Sistema Operativo superior

El sistema operativo GNU/Linux es la respuesta de programadores independientes al monopolio de software de baja calidad y masivamente popular de Microsoft Windows. Tan popular que la mayoria de la gente piensa que Windows es parte de la computadora, y no una pieza que se puede reemplazar, quizas por algo de mejor calidad. Miles de programadores fueron creando distintas partes de un sistema operativo libre llamado GNU, al que le agregaron el kernel (o núcleo) llamado Linux. Qué es Linux?

Linux server - 2005-11-19

En la empresa que trabajo finalmente compraron una computadora para correr Linux. Un server Dell de 64 bits con un micro AMD, 1GB ram y 120GB de HD. Vino sin OS, a pedido. Instalé Linux CentOS 3, que era sobre lo que el software de Xilinx instala correctamente. Le fui agregando varios servicios, además de ser usado como server farm para compilar firmware. Instalé un samba server para que los clientes Windows copien sus archivos a compilar. El print server de samba resultó ser extremadamente estable y rápido. Kudos to samba. Tambien está corriendo un servidor Subversion para excelente source code version control o control de versionamiento de código fuente.

Experiencias del mundo real - 08/21/03

Ernie Ball es una compania que fabrica cuerdas de guitarra, desde San Luis Obispo, California. Hace tres años, la BSA (la version de Soft Legal americana) toco a su puerta, sin aviso, con oficiales de policia y procedieron a registrar su empresa. Sterling Ball, su CEO, paga por el software en sus computadoras, en su mayoria Microsoft cada vez que compra una computadora. El pensó que no tendria problemas; se equivocó. Encontraron que un 8% de las 72 computadoras tenian problemas de licenciamiento. Terminó pagando $100000 (cien mil) dólares de multa (25000 por gastos de corte) a Microsoft por usar estas licencias sin permiso. No hubiese sido tan malo, sino hasta ver que el mismo Microsoft empezo a usar esta empresa para demostrar que es necesario pagar por licenciamiento y su empresa aparecio como "infractora" en diarios y television, como parte de la maquina de marketing de Microsoft. Realmente humillante. Ahi es cuando Sterling Ball decidio pasarse 100% al Open Source y no pagar ni un centavo mas por software a Microsoft. La noticia original en news.com en ingles.
